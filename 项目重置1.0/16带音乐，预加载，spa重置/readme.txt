文件夹说明：
audio 音乐
css 样式
font 字体
images 图片

js 
  jquery.js  js库
  global.js 公共js
  friend.js 朋友接收页面的js
  vispa.js 单页面的js
  vispa.config.js 单页面的配置信息 内含目录页，首页（默认进入的），编辑页
  app/
    edit.js 编辑贺卡页面的js
    index.js 无交互内容，但是单页面配置的时候必须有
    main.js 无交互内容，但是单页面配置的时候必须有
view（和 vispa.config.js 里面的配置对应）
    main.html  首页（这是默认的）
