window和document的区别？

1.window指窗体，document指的是页面;document是window的一个子对象
2.用户不能改变document.location，但可以改变window.location,
3.window.location是一个对象，document.location不是。

["1","2","3"].map(parseInt) 的答案是多少？

  [1,NaN,NaN]
因为 parseInt 需要两个参数(val,radix)，其中 radix 表示解析时用的基数。
map 传了3个(element,index,array)，对应的 radix 不合法导致解析失败。

//答案
 var parseInt = function(string) {
    return Number(string);
    };
    console.log(["1","2","3"].map(parseInt));

jQuery 与 jQuery UI 有何区别?

`jQuery`是一个js库，主要提供的功能是选择器，属性修改和事件绑定等等。
`jQuery UI`则是在jQuery的基础上，利用jQuery的扩展性，设计的插件。提供了一些常用的界面元素， 诸如对话框、拖动、自动补全，折叠面板，排序等等
